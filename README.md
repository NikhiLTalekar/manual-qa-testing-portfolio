# manual-qa-testing-portfolio

End-to-End Manual & Exploratory Testing of an E-Commerce Web Application

Project Overview
This project demonstrates end-to-end manual and exploratory testing of a web-based e-commerce application. The objective is to ensure the application functions as expected, delivers a good user experience, and handles edge cases and validation scenarios effectively.

The project focuses on real-world QA practices, including test case design, test execution, exploratory testing, and defect reporting.
